## About this project

This project is a panda management system. It is a full stack project that uses the MERN stack (MongoDB, Express, React, Node.js).

![Happy Panda](panda.jpg)

### Features

- Create, Read, Update, Delete (CRUD) operations
- Search
- Pagination
- Filtering
